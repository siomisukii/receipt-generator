# 領収書ジェネレーター

スマホからすぐ領収書PNGを発行できる、単一HTMLアプリ。

## 使い方

1. 「設定」を開いて宛名・発行者名を入力（端末に保存される）
2. 金額を入れて「発行」
3. 「PNG保存」でダウンロード（iOSは新規タブで長押し保存）

## 仕様

- 但し書き: `家賃として`（固定）
- 領収書番号: `YYYYMMDD-XXX`（同日内で連番）
- 日付: 発行ボタン押下時の現在日

## プライバシー

- 入力した名前・金額は **すべて端末のlocalStorageに保存** され、サーバーには一切送信されません
- 領収書PNGも端末上で生成されます

## 技術

- Vanilla HTML / CSS / JS
- 依存ライブラリは [html2canvas](https://html2canvas.hertzen.com/) のみ（インライン埋め込み）
